## 10.19 iptables规则备份和恢复
* 关键在于 /etc/sysconfig/iptables 这个配置文件
```
service iptables save //会把规则保存到/etc/sysconfig/iptables
iptables-save > my.ipt  把/etc/sysconfig/iptables的规则备份到当前目录下的my.ipt
iptables-restore < my.ipt  恢复
```
